# 1. Abrir el repositorio de la aplicación
Lo primero que deberemo hacer es abrir el repositorio de la aplicación el cual se encuentra en el siguiente enlace:
https://github.com/ErDanieh/WIKI.IMCR

# 2. Acceder a la carpeta pages del repositorio
![image](Images/ImagenesTuto/AccederPages.png)

# 3. Crear una nueva página
Aqui podremos crear una carpeta desde la barra de navegación y dentro de esta carpeta crear un archivo con la extensión .mdx
o directamente crear un archivo con la extensión .mdx para crear una página sin carpeta. 
![image](Images/ImagenesTuto/CrearPagina.png)

# 4. Crear el contenido de la página
En este paso deberemos crear el contenido de la página, para esto deberemos seguir las siguientes instrucciones:
- Para crear un título deberemos usar el símbolo # seguido del texto que queremos que sea el título.
- Para crear un subtítulo deberemos usar el símbolo ## seguido del texto que queremos que sea el subtítulo.
- Para crear un subsubtítulo deberemos usar el símbolo ### seguido del texto que queremos que sea el subsubtítulo.
- Para crear un párrafo deberemos usar el símbolo > seguido del texto que queremos que sea el párrafo.
- Para crear una lista deberemos usar el símbolo * seguido del texto que queremos que sea el elemento de la lista.
- Para crear una imagen deberemos usar el símbolo ! seguido del texto que queremos que sea el texto alternativo de la imagen y entre corchetes la ruta de la imagen.
- Para crear un enlace deberemos usar el símbolo [ seguido del texto que queremos que sea el texto del enlace y entre corchetes la ruta del enlace.
## Agregar una imagen
Para agregar una imagen deberemos subirla a la carpeta Images y luego usar la ruta de la imagen para agregarla a la página.
![image](Images/ImagenesTuto/AgregarImagen.png)

## IMPORTANTE PONER LA EXTENSION DEL ARCHIVO .mdx
![image](Images/ImagenesTuto/CrearContenido.png)

# 5. Guardar los cambios y proponer el contenido
Una vez que hayamos creado el contenido de la página deberemos guardar los cambios y proponer el contenido para que sea revisado por el administrador del repositorio.
![image](Images/ImagenesTuto/GuardarCambios.png)

# 6. Crear un pull request
Una vez que hayamos guardado los cambios deberemos crear un pull request para que el administrador del repositorio pueda revisar el contenido y aprobarlo.
![image](Images/ImagenesTuto/CrearPullRequest.png)

Una vez se envie la pull request Vercel se encargará de verificar que no se haya roto nada
![image](Images/ImagenesTuto/Verificando.png)

# 7. Ver el contenido en la página
Una vez que el administrador del repositorio haya aprobado el contenido de la página podremos ver el contenido en la página.
[https://wiki.imcr.vercel.app/](https://wiki.imcr.vercel.app/)
